# dotfiles
> preferences and stuff

#### 1. Symlinks

[RCM](https://github.com/thoughtbot/rcm/blob/master/README.md) will handle most of the work for us.
```sh
# Dry run to make sure everything looks ok:
$ lsrc

# Create symlinks
$ rcup -v
```

#### 2. Download and Install
1. [iTerm](https://www.iterm2.com/)
2. [Karabiner](https://pqrs.org/osx/karabiner/)
3. [Brew](https://brew.sh/)
4. [Docker](https://www.docker.com/docker-mac)

#### 3. Manual Stuff
1. Have iTerm load our preferences file.

    Open `iTerm -> Preferences -> General -> Preferences`, and select the `.dotfiles` directory.

2. Copy the fonts to the relevant location.
